DISCIPLINA WEB II

Em HTML, vemos como temos toda uma estruturação de como será nosso site (tittle, head, meta, body), com esses códigos podemos criar o "corpo" do site, fazendo o mesmo por partes.

HTML é uma linguagem de marcação, ou seja, não é usada para programar, mas sim como um complemento para o seu código já existente. HTML - 5 é a última versão lançada e consequentemente, a mais atual. Com HTML podemos criar a estrutura do site web que está em desenvolvimento e assim fazer a sua "espinha dorsal".

CSS

CSS é uma linguagem de marcação também, mas o seu propósito é apenas estilizar o HTML já existente.

Com CSS podemos editar cores, fontes, formatação de textos, tudo relacionado ao design do site em HTML já existente.

PHP

PHP é uma linguagem de desenvolvimento web, podendo ser complementada com HTML5 e CSS.

PHP é uma linguagem de desenvolvimento web criada exatamente para este propósito. É rápida, flexível e pragmática. PHP contém tudo o que você precisa para seu site e também para os mais populares sites do mundo.

Banco de Dados - MySQL

Banco de Dados é onde armazenamos os dados dos usuários que visitam ou já visitaram o site em questão.

Banco de Dados - MySQL, é onde armazenamos os dados dos usuários que frequentam ou já visitaram o site. Armazenam dados como cadastros, telefone, e-mail, tudo relacionado ao usuário fica armazenado no Banco de Dados, mas para termos onde armazenarmos estes dados precisamos alugar um servidor, até mesmo para o site poder ficar online e visível para as pessoas visitarem.
